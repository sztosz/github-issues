    format_for(column_widths)

    Return a format string that hard codes the widths of a set of columns. We put " | " between each column.

    Example

    iex> widths = [5,6,99] iex> Issues.TableView.format_for(widths) “~-5s | ~-6s | ~-99s~n”

    print_table_for_columns(rows, headers)

    Takse a list of row data, where eacg row is a Map, and a list of headers, Pronts a table to STDOUT of the data from eacg row identified by each header. That is, eacg header identifies a column, and those columns are extracted and printed from the rows.0

    We calculate the width of each column to fit the longest element in that column.
